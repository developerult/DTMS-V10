Warehouse Scheduler

 

Watch A Video

 

The user document is intended to help and assist the users in using the scheduler for setting dock appointments and its related features of this module. This document provides screenshots of the relevant functionality and explains in detail about the available functionalities.

 

The Warehouse scheduler page provides the ability for authorized users to view the schedule for the selected warehouse. The scheduler page shows the Calendar with different views (daily, weekly, monthly and timeline) to see already scheduled appointments for the orders.

Additional set up may be required based on your system configuration. If you desire to turn on Carrier Scheduling Automation, additional set up is required at the Legal Entity level as well as the carrier level.
