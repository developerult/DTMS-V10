Edit Appointment Calculation Time Factor

1.  The user can edit details of an existing appointment calculation time factor by clicking on edit  button available for each appointment calculation time factor in the list.

2.  Admin can click on edit button for the popup to open in edit mode as seen in the screenshot below.

3.  Once the details are modified, the user should click on “save” button to save the changes or close popup to cancel the changes.
