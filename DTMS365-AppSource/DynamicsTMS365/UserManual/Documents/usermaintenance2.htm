User Maintenance

Users must now be maintained in the D365 User Maintenance page.  Administrators can Add, Edit, Delete or Deactivate user accounts.  Use the Action menu to perform the following:

•      Change Legal Entity:  Select the legal entity where users are assigned.

•      Assign User LE:  Provides the ability to move a user to a different Legal Entity.

•      Add New User: Add a new TMS, Warehouse, or CSR Sales users.  Do not use this option to add a Carrier User.

•      Add New Carrier User: This option provides the details needed to set up a Carrier login to NEXTrack, select the carrier, and if this user has access to the accounting (Settlement) pages.

•      Manage Roles:  This screen provides visibility to Screen, Report, and Procedure settings for each Role.  It has some basic abilities to manage settings.  For detailed or complex changes, we recommend using the Desktop Client to manage Role and User specific Security if the feature you are looking for does not exist on this page.

•      Assign Role LE:  provides the ability to modify the Legal Entity setting for a Role.  It is recommended that you contact NGL Support for assistance when you wish to use this page.

•      Mass Update User SSOAs: we now support the ability to connect to various remote systems; like DAT, API, Bing Maps etc.  this list will grow over time.  Users must be configured with credentials for connecting to these resources.  The Mass Update dialog is a utility to update multiple settings at one time.

•      Manage SSOA Settings: This is were you set up the Single Sign On settings for each remote system that are shared by all users, like the connecting URL.  Please contact NGL support before making changes to these settings.
