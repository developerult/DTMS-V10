System

 

Dynamics TMS System Page is only given access to system administrators for configuration maintenance and user maintenance. This is also where legal entities are managed.

Only users given ADMIN credentials will have visibility of this page. The Admin access will allow for some system configuration for adding legal entities and turning on the Scheduling (Appointment Management System) Carrier Automation.
