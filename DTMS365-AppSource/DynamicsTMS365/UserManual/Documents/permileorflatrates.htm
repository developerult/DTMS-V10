Per Mile or Flat Rates

Can be configured as:

1) Country Code is always required
2) State/Prov rates (Country code and St/Prov required)
3) City rates (Country code and State/Prov required)
4) 3 digit or 5-digit (Start/Stop) postal code ranges (Country code and ST/Prov required)
5) Lane specific-enter in the lane number from TMS Lane Maintenance for a rate to that destination.
