Find My Appointment

The Find My Appointment feature allows the user to search for the appointments based on the selected search criteria with the appointments satisfying the criteria listed in the grid.

1.     Select one of the search criteria from the dropdown in Find Appointment section as seen in the screenshot below:

 

 

2. Input appropriate search value in the input box and search by clicking filter button.

3. The search results matching the search criteria are displayed in the grid and can also be seen in the calendar, as seen in the screenshot below.
