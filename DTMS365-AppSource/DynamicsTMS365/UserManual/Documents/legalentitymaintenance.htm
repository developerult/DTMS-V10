Legal Entity Maintenance

This is an Administration page and not intended for all users. Legal Entity Maintenance page will store and manage all legal entities on the system and their configuration. Before adding a new Legal Entity, Admins should check with support before making changes.      

 

 

 

 

 

To edit a Legal Entity, click on the pencil icon. Users can set default BOL Legal text and Dispatch Legal text for when using web client BOL. Additional settings for the Carrier Appointment Management Automation and Tender email notifications.

Carrier Accept Load Mins = The amount of time in minutes carriers have to accept a load tender. This can be set at the Legal Entity level or carrier specific level. See Carrier and the specific carrier to configure separatley.

Expired Loads Email To = Email address used to receive expired load tenders.

Expired Loads Cc = Carbon copy Email address used to receive expired load tenders.

Carrier Appt = On/Off. When off the carriers are not allow to self-schedule appointments in the scheduling system.

Allow Car Appt Edit = On/Off. When on, allows carriers to edit their appointments using the automated scheduling system. When off, it requires a request to be made via the carrier through the scheduling system.

Allow Car Appt Delete = On/Off. When on, allows the carrier to delete their appointment without any approval. When off, it requires a request to be made via the carrier through the scheduling system.

Appt Mod Cut Off Mins = The time in minutes, a carrier must appoint an order before a request needs to be made by carrier. For example, 1440 minutes means carriers must appoint orders before the 24-hr. window load date.

Def Last Load Time = The time designator for the last appointment available to be made. Meaning, the system will prevent a carrier using automation for scheduling appointments beyond this time.

If you intend to use the Carrier Self Scheduling Automation, you must turn on the automation configuration by Legal Entity. This will allow all associated warehouse locations to this legal entity to be configured for scheduling automation.

Click and highlight the Legal Entity, then click on Set AMS Carrier Automation.

This Legal Entity and associated warehouse locations can now use the automation. NOTE, if you want to turn OFF scheduling automation at the warehouse level, this must be done by Warehouse. Go to the Company page, expand the black arrow, and click on Warehouse Carrier Settings to turn off automation. Each carrier will need to be added in order to not have access to use the automation. See DTMS How to Turn On/Off Scheduling Automation documentation for more information.

NOTE: Additional Warehouse location set up is required.
