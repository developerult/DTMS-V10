Desktop Import

 

To import a fuel matrix, users must use the fuel matrix template using excel. All fuel matrixes should be maintained in excel format. TMS will only utilize the first 5 columns.

First row = Effective Date (date cells must be formatted as a date)
Second row = Low range (best to use two decimal places)
Third row = High range (best to use three decimal or more places to prevent overlapping)
Fourth row = Per mile cost
Fifth row = Percentage cost

 

1) Prepare your fuel matrix in excel for import. Following the layout above.

 

 

 

 

2) Go to the Fuel Addendum page in the Desktop client for your carrier.

3) Copy the cell columns from your excel file.

4) Place your cursor next to the PLUS icon (red below) and Paste the information using Control V.

The fuel matrix will load in 5 – 10 seconds and is ready to be imported.

8) Click on Fuel in the Actions menu and you will see the fuel set up you already created.

9) Click on Fill From Template in the Actions menu.

The Fill from Template dialog box will open.

10) Turn ON use specific Carrier for Template. If you do not see the available carriers drop down list. Try refreshing your screen.

11) Select the carrier you want to copy (One loaded from the desktop client).

12) Click Save – Then Refresh. The matrix will load. Click Save.

Any time there is an update to the fuel matrix, users should click on Update All Active Loads, so the system will update current fuel based on changes. Best to do it early in the morning or end of day as process can take some time.

        

You can now delete the fuel matrix in the desktop client. TMS will use the tariff fuel matrix as the default calculation.
