Calculating Appointment Time Using Calculation Time Factor Rules

Check the Company Parameters for DefaultWeightUOM and DefaultCubeUOM
Check if we have a CTFR for Weight for UOM = DefaultWeightUOM
Check if we have a CTFR for Cube for UOM = DefaultCubeUOM
Check if we have a CTFR for Quantity
See if we have CTFRs for Package Type

For each rule that exists and matches the load calculate the total minutes

Out of the 4 rule totals use whichever is the greatest (max)
If the total calculated minutes is less than the configured minimum then we use the minimum
