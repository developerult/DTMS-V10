Finalized Bids

Accepted bids will move to the Finalize tab and display a status code of Accepted, meaning the TMS user has accepted this bid and it has been tendered to the carrier.

To see the status of the load, go to the Load Board and apply your filter to show all loads in PC (tendered) status. When a user accepts the bid the system will move the workflow to PC status (tendered) automatically.

 

When carrier accepts the load, the system will automatically set it to PB status (finalized-meaning carrier accepted the load). Note, the carrier can still reject the load, in which case will set the status to N. User will need to re-book or post load again.
