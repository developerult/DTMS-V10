New Resource

1.  Copy the resource settings for the new resource by entering the new ‘Resource ID’ and ‘Resource Name’

2.  Once the ‘Resource ID’ and ‘Resource Name’ is entered, admin can click on “Add Resource and Copy Settings” button for create the new resource and copying the settings.

3.  User can override the settings of an existing resource by selecting another existing resource to the copy settings.
