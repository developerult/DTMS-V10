Delete a Contract

Upon creating a contract and if it is determined a mistake has been made and you would like to delete and remove the contract.   

Select the contract so it is highlighted and click the delete button.

If a contract is in use and has been assigned to orders/loads, the only way to make the contract inactive is to insert an expiration date. Edit the contract by clicking the pencil icon and insert the expiration date. Then you can create a new contract.
