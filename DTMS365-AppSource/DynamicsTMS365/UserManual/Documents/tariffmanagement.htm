Tariff Management

 

The tariff management system maintains carrier contracts and rates for each warehouse location. This can be inbound and outbound tariffs.

Outbound tariffs will contain the origin company location and the rates are to the various destination points whether they’re state/prov, city, or postal code specific. Must include country code.

Inbound tariffs will contain various origin points whether they’re state, city, or postal code specific to a company location. Must include country code.

The country code must match that of the general system set up for Origin / Destination locations.
