Package Descriptions

 

Package Description allows users to preconfigured Product Descriptions, Class, and NMFC number to select during rating when using the Web Client. This feature allows users to easily select the package description and the system will auto insert the description, class, and NMFC number rather than having to manually make an entry with each order.

To add a new description, click on Add.

Insert the Name, Description, FAK Class, NMFC and if necessary NMFC Sub Class. Click Save.

 

Note, if the NMFC and NMFC Sub are incorrect, the system will produce an error during rating. Be sure you have the most current information loaded.

 

 

When ready to Rate It, simple click the drop-down arrow for Pkg Desc. and select the description and the system will auto insert the description, class, and NMFC number. Now just insert your dimensions, weight and pallet count.
