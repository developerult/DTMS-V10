Exporting a Tariff Rates

To export, go to the Systems Tab in the Desktop client and click on Tariff Import Tool found in the Systems tab on the desktop client.

 

The Tariff Import Management screen will pop open.

Begin by using the selection arrow to select the Carrier.

Select the warehouse location for the tariff assignment.

Select the most recent contract you wish to export. (Note, the bottom one is the most recent)

Select the Equipment associated with this contract.

Once you have selected the equipment, the system will auto-fill the tariff type.

•      Rate Name

•      Class Type

•      Rate Type

•      Bracket Type

Click (Optional) Export Rates to Excel

The system will produce the rates from this tariff into an excel format. Note, users should not manipulate the columns as the system will only follow the format exported. Do not change the format when importing rates.
