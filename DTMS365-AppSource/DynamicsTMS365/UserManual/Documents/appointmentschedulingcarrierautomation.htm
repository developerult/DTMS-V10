Appointment Scheduling- Carrier Automation

 

Carriers will only have access to this page if the self-appointed automated scheduling system has been configured. Carriers will have access to appoint both pickup and delivery appointments for loads that have been assigned to them. Using your normal login credentials, carriers can click on Carrier or Carrier Grouped under Appointment Scheduling to see all orders/loads ready to appoint.

Carrier – Will display a single order associated with a ShipID (CNS Number)

Carrier Grouped -expand the black arrow to display all orders linked to the same ShipID. An example would be a multi stop shipment with many orders that have the same ShipID.

 

The pending orders summary chart will display orders that are ready to appoint within a 72-hour window (Time window based on user configuration). Carriers can appoint orders/loads as soon as the load is available under the Unscheduled Orders – Pick Up tab.

To schedule an appointment, click on the Unscheduled Orders tab.

Here you will see a list of the orders/loads that have been assigned to you and waiting to be appointed.

Grouping- Users can group the data by CNS or ShipID by clicking on the label and dragging the column up and releasing in the block. This will group all orders assigned to the same CNS or ShipID for easier viewing. In other words, the same load.

 

 

 

 

 

 

 

 

 

To make an appointment, click on Book Appt. A dialog box will open displaying the warehouse location and times available to select. Use the scroll forward to find additional time slots. Carriers should always use a time in the future when making appointments.

Book Appt = Select to confirm this time slot.

Warehouse = Location where the load is to be picked up.

Date = Appointment load date

Start Time = Appointment time.

Use the arrow to page through available time slots.

An email confirmation will be sent to the carrier scheduler email address that was configured.

 

If you have multiple orders to be appointed and want to assign them to the same Appointment Id and time, use the Equipment ID to apply to multiple orders.

 Click on the pencil icon, the Equip ID field will open. Enter in a unique identifier that will link multiple orders to the same appointment. For example, could be a trailer number, carrier number etc. Apply this identifier/number to all orders in order to have the same pick up appointment applied.

click on Select Appt. A dialog box will open displaying the warehouse location and times available to select. Use the scroll forward to find additional time slots.

To see appointments that have already been booked-click on Booked Appointments.

To edit or delete an appointment, click on the Pencil icon to edit or Click on the Delete to delete the appointment. Depending on the system configuration, shippers may allow carriers to edit/delete or they will have to make a request via the system to edit/delete an appointment.

Carriers may not be allowed to change an appointment if the request is within a certain time frame prior to loading based on shipper’s configuration.
