Temperature Types

1.  The temperature types for a resource can be setup and managed by clicking the edit  button available in top header of Temperature Type list grid.

2.  Admin can click on edit button to open the popup window listing the temperature type so that the supported temperature type can be selected from the grid.

3.  Once the admin has completed selecting the temperature types to be associated, they can click on “save” button to save the changes or click to close popup button to cancel the chosen temperature types.
