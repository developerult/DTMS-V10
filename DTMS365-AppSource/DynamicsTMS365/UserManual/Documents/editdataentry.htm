EDIT – Data Entry

To EDIT information about a load/order, provide status updates, click on the pencil icon.

The Edit screen will pop up. Be sure to click the save icon in the upper right-hand corner of the screen to save entries.

Expand the arrow to enter Pick Up information if not already expanded.

Expand Delivery to enter Delivery information if not already expanded.

Carriers who will be using the Settlement page to enter freight invoices MUST enter in the Delivered Checked in Date & Time information highlighted in yellow above. Otherwise, the transaction will not be available for invoice entry in the Settlement page.

 

Expand Trailer/Load information if not already expanded.

Expand Notes to enter in notes, comments, and select reason code for late deliveries.
