Desktop Client Set Up


When using the Web Client, the final step to complete your carrier set up is to open the Desktop Client of Dynamics TMS and complete the set up in the Carrier Maintenance Screen.

Click Carrier Tab, search by Carrier Name using the drop-down list.

Insert Carrier SCAC Code.

Click on Qualification and insert the carriers Insurance Expiration Date.

Check the box on Qualified.

Maximum Desired Exposure Per Shipment, should equal the amount of the carrier’s cargo insurance.

Maximum Desired Exposure All Shipments, should equal a larger value based on total number of open loads at a time. $1,000,000 is often used.

Click on Main and click Save.

 

Next add carrier tariffs, fuel, and configure carrier accessorials tolerances in the Carrier page. This should be done for each carrier. This allows for fees that are entered via the portal or EDI to follow the automated approval process.
