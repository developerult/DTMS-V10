Edit-Data Entry

Click on the pencil icon to see or enter additional tracking fields such as Pick Up and Delivery appointment information, trailer number, seal number and more.

When using the scheduling system, the date and time will auto populate. The appointment fields will also be grayed out while using the scheduling system and once an appointment is created. This is to protect the appointment for use within the scheduling system and that no one can edit appointments when schedule is in use. Upon delivery, the fields will no longer be grayed out. Trailer/Load information is a manual input.

 

 

The Edit screen will pop up. Be sure to click the save icon in the upper right-hand corner of the screen to save entries.

Click and expand the arrow (yellow highlight) to enter Pick Up information if not already expanded.

Click and expand arrow for Delivery to enter Delivery information if not already expanded. NOTE: Delivered-Checked In Date/Time is when the order/load actually delivered. This is the field that tracks on time performance as well as the field that needs to be populated for carriers to enter in freight bills.

Click and expand Trailer/Load information if not already expanded to enter Trailer/Load information.

Click and expand Notes to enter in notes, comments, and select reason code for late deliveries. Once entered, this information is stored in the system for all users to view when they tract load status.
