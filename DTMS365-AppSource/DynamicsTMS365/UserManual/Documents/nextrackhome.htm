Login page

 

A BLUE question mark can be found on each page you are working on. Here users can store information and notes about a particular page and edit the content.

 

 

 

Click on the Pencil icon to edit. After entering in notes, be sure to click the save button. Entering notes by a specific user, will only be seen by that user. System Administrators can enter in system wide notes for all users.

To change the column headers and select the data preferences to view, look for this icon to check on/off to display data. This is available anywhere you see this icon on a page.
