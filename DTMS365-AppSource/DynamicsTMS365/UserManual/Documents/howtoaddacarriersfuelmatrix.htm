How to Add a Carrier’s Fuel Matrix

 

A carrier specific or shipper specific Fuel Matrix can be created for each contract. Fuel can be managed in two places. 1) The desktop Carrier Maintenance Screen 2) The Contract level. It’s recommended to insert the fuel matrix at the contract level in case you come across a carrier with multiple fuel matrices by mode.

 

1) Go to Contracts and highlight the carrier you want to create a fuel matrix for.

2) In the Actions menu, click on Fuel.

3) The Fuel Addendum page will open.


 

4) Turn ON Use Nat. Avg for Updates.

This section lists two possible selections for an average fuel cost source. They are:

•      Use National Average for Update = Turn ON to use National Average.

•      Use Carrier Default Fuel Price = Blank

 

        

5) Then Select an Exception Processing Method

          Use National Zone Average for Each State = Turn ON (only if you intend to use zones like Midwest) or Turn OFF otherwise to use Exception List for Specific States. (default)

6) Then select A Revenue Calculation Method.

ON = For Rate per mile
OFF = For Percentage

Click Save.

You have just prepared the fuel matrix for loading.
