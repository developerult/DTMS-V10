Rates

TMS supports various types of rates and calculations. Some examples include:
1) Per mile rates
2) Flat rates
3) $/lb. rates (CWT is not supported and should be converted to rate per pound)
4) Class rates
5) Unit of measure rates (Per pallet, cube etc.)

NOTE: All tariffs need to include country code and match the country code used on the origin/destination pairing on the order, otherwise, system will not find a tariff to calculate rate. Also, spelling of city name is important or the system will not find the correct rate.
