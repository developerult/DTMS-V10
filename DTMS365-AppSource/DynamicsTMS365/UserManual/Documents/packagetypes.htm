Package types

1.   The package types supported type by particular resource can also be setup by clicking on edit  button available in the toolbar of package Type list grid.

2.   The admin can click on edit button for the popup window to open and list the package types so that admin can select the package type supported.

3.   The selected package types are associated to the resource once the admin clicks on “save” button to save the changes or click to close popup button to cancel the selected Package types.
