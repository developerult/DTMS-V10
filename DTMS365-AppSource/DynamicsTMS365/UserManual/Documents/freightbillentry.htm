Freight Bill Entry

 

When set up to enter freight invoices into TMS, specific TMS users can be assigned access to enter in invoices by carrier.

Go to the User Maintenance Page in System for the corresponding Legal Entity and select an internal user to grant access to the carrier’s view of the invoice transactions.

Click on Associated Carriers tab.

Click on Add.

Select the carrier to be associated with this user and select the Accounting Check Box. Do this for every carrier you want to be associated with this TMS user. Note, when adding a new TMS users, this set up can be done upon the creation of the new user.

 

This will now provide access to the web settlement screens and allow this user(s) to enter in carrier freight invoices manually. Note, users will need access to the Settlement Page as part of their Role access.
