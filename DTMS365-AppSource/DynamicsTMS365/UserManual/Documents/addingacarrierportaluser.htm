Adding a Carrier Portal User


Adding carrier users will allow access to Web tenders, Load postings, Online scheduling, Freight bill entry. We have two type of roles for carrier users. If your configuration will not use any of the above features, you do not need to set up carrier users. Only Administrators can add users to TMS.

1) Carriers - This user profile is typically for a dispatcher (accepts loads, schedules appts, and provides load status.
2) Carrier Accountants – This user profile enters in freight bills into the website but also can be a dispatcher (accept tenders) and other responsibilities like above.

NOTE: If you have dispatchers who also are responsible for entering in freight bill data, you can just set them up as a Carrier Accountants. Both carrier set ups will have access to the tender page.

In the Systems page, click on User Maintenance. Be sure to select the correct legal entity you want to work in if you have multiple legal entities.

In the Actions section, click on Add New Carrier User

Enter in carrier’s email address. This email address is the address that the TMS system will use to communicate with this user.

A recommended username is the carrier contact email address. It’s also recommended that you manually create a password and share it with the carrier directly. Before doing so, users should login as the carrier to confirm setup is correct.

Check the box for accounting ONLY if this user will be entering in freight invoices. This will give the user access to the Settlement and Settled pages.

Use the drop-down list to select the carrier this user is associated with.

Click Save in the upper right-hand corner.

An email will be sent to this carrier contact to include their password. Carriers can change their password once they login and by going to settings.
