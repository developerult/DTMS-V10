Carrier Confirmation/Notification

Carrier Confirmation emails will be sent to the carrier whenever they create, update, or delete an appointment via the Self-Service Portal (Confirmations will not be sent when they send create/update/delete requests to the warehouse users)
Carrier Notification emails will be sent to the appropriate carrier when the Warehouse User creates, updates, or deletes an appointment via the Warehouse Scheduler

Booked Confirmation - Confirms the Carrier booked an appointment via the Self-Service Portal


Modified Confirmation - Confirms the Carrier modified an appointment via the Self-Service Portal

Deleted Confirmation - Confirms the Carrier deleted an appointment via the Self-Service Portal


Booked Notification - The Warehouse User booked an appointment via the Warehouse Scheduler


Modified Notification - The Warehouse User modified an appointment via the Warehouse Scheduler


Deleted Notification - The Warehouse User deleted an appointment via the Warehouse Scheduler
