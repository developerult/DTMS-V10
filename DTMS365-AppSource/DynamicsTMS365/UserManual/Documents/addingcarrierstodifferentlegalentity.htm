Adding Carriers to Different Legal Entity

If you have multiple Legal Entities, be sure to assign the appropriate carrier(s) to each Legal Entity. If you use the same carriers across multiple Legal Entities, users can simply import the same configuration from one LE to the other by clicking on Import Existing LE Config.

 

Select the Legal Entity you want to import from in the drop-down list.
Click Import.

The system will copy the same carrier configuration from one LE to the other.
