Calculation Time Factors

There are 4 Calculation Types - Quantity, Weight, Package, and Cube
Only one rule is allowed for type Quantity (there is no UOM for Quantity)
Types Weight, Package, and Cube can each have multiple rules (one for each UOM)

Note: If no Calculation Time Factors are provided then the Carrier Automation algorithm will use the Average minutes as the appointment length. 
Also, all package types from the load must match a Calculation Time Factor Rule or else we use the Average minutes as the appointment length

Fields
Amount - The number of units of (ex 500lbs)
Time Factor - The number of minutes it takes to unload the Amount (ex 10 mins) --> 500lbs/10mins

Example
Calc Type = Weight; UOM = LBS; Amount = 500; Time Factor = 10;
This is saying that this dock can load/unload 500 pounds every 10 minutes (500lbs/10mins).
So a load of 2000 lbs. would take 40 minutes to load/unload.
