Rate Shopping

Rate shipping is intended to provide support for planning and cost estimation, it also provides TMS Users the ability to quote and dispatch loads without the need for order integration with external applications.

The NGL TMS API provides the ability to compare rates and quotes from multiple sources.  Including the following:

•      Real time API integration with LTL Carriers

•      Real time cost analysis with private tariffs for carriers that do not support API integration

TMS user enter or edit the load details, including the following:

•      Pickup and delivery information may be typed or edited for previous quotes or selected from an address book.

•      Package and class types

•      Dimensions

•      Accessorial Fees and Fuel

Once all the data has been entered user click to “Rate IT”

When rates are returned TMS Users can filter the results, like show me rates between $100.00 and $500.00.  Details for each quote may be expanded to display cost details like fuel, line haul and discounts.  The selected quote may be dispatched to the provider with the “Book It” option. 

 

 

Before a load is dispatched TMS users have the ability to add additional information to the carrier like contact information, pickup or delivery notes, and order specific identifiers like Order Number or PO Numbers.  Once the information is complete a simple click with dispatch the load.  TMS will create a booking transaction record and store the selected carrier, quote and dispatch information.

 

After dispatching TMS Users may print, or save as pdf, the displayed Dispatch Report and the Bill of Lading. 

 

 

Previously generated Quotes are generated and stored as a historical record and may be dispatched, requoted, or used as a template for future shipments.  It is also possible to dispatch the load from the Historical Quotes “Book It” option.   Filters may be applied to the results like Origin Name to assist with selection.
