NEXTStop Ops

 

This feature facilitates the process of managing all loads that have been submitted for bid to your internal carrier base already set up in TMS. Carriers that have been granted access to the Carrier portal system, will have visibility of all loads to bid on in their NEXTStop Carrier screens. This access does require carrier users to be set up and the Load Posting Carrier setup. See your System Admin on setting up Load Posted Carrier and New Carrier User access.
