Pallet Based Rates

Can be configured as:

1) Country Code required
2) State rates
3) City rates (Country code and State/Prov required)
4) 3 digit or 5-digit (Start/Stop) postal code ranges (Country code required)
5) Lane specific-enter in the lane number from TMS Lane Maintenance for a rate to that destination.

When using pallet-based rates, be sure to configure your pallet breaks correctly so the proper calculation is done.
