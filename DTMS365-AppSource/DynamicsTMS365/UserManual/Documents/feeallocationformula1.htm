Fee Allocation Formula

Fee allocation formula uses allocation bracket type to determine what metric value is used in the allocation calculation

1.  Pallets

2.  Volume

3.  Quantity

4.  Weight; Pounds; CWT (hundred weight)

5.  Distance

6.  Even = allocate evenly by # of orders defined by the fee allocation type For Example: if the fee allocation type is destination, and there are 2 orders being delivered to the same location, an even allocation formula would allocate 50% of the fee value to each order
