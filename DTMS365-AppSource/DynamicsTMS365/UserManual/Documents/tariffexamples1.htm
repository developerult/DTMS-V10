Tariff Examples:

LTL Weight Based Tariff.
Use column J thru S for the rate value. Note rate values must be in cost per pound and not CWT.

LTL Class (Class must be populated) Weight Based Tariff.
Use column J thru S for rate value. Note rate values must be in cost per pound and not CWT.

Truckload Distance Tariff.
Use column J for the rate value.

Truckload Flat Tariff.
Use Column J for the rate value.

Pallet Based Tariff.
Use column J thru S for the rate value.

Note the ranges between pallet brackets. If there was an order for 2 - 4 pallets, the system will select the 2-pallet rate. For pallets 5-7, the system will rate each at the 5-pallet rate.
