Carrier Automation Configuration Fields

The below fields are only editable when Carrier Automation is turned on for the Legal Entity.

Allow Carrier Appt Edit - When off carrier cannot edit manually and is forced to send an email request
Allow Carrier Delete - When off carrier cannot delete manually and is forced to send an email request
Appt Mod Cut-off Mins - Number of minutes before the appointment time after which the carrier can no longer makes changes manually and is forced to send an email request
Default Last Load Time - Used only for calculating if the cut off time has been met. This value will be used when a time value has not been entered for the Load/Ship date. Note: A value of 12:00 am in the Load Date will be considered as not entered.
Appt Not Set Alert Mins - This value is not currently used
