How to Turn Off Scheduling Automation by Warehouse

1) In order to turn off scheduling automation by warehouse, go to the Company page in the Web Client.
2) Find the warehouse location and expand the black arrow.
3) Click on Warehouse Carrier Settings



Users will now be able to turn off the scheduling automation by carrier. This will have to be done for each NON automated warehouse.

4) Click on Add and insert each carrier.
5) Turn OFF -Carrier Appt Automation, Turn OFF Allow Car Appt Edit, and Turn OFF Allow Car Appt Delete.

Carriers will still have visibility of these orders to appt, however, if they try to schedule an appointment, they will only be able to Request an Appointment. This request will be sent to the TMS scheduling system email that was created to receive all scheduling emails.

Another way to Turn Off Scheduling Automation at the warehouse location is to Turn Off Validation for each dock door resource.

When OFF, system will not apply the algorithms of the automation and carriers will not have visibility of this dock door (resource). Nor will the system adhere to the Override settings and additional settings such as Temperature Settings, Package Type, or Time Factor.

 

Therefore, it’s recommended to Turn OFF the automation at the warehouse location by adding each carrier.
