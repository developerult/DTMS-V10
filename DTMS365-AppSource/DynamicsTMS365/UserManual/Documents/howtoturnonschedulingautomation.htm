How to Turn On Scheduling Automation

 

The Scheduling Automation ON/OFF is set at the Legal Entity level. When ON, all warehouse locations will be using the Carrier Scheduling Automation features and default settings.

1) Go to System – Legal Entities
2) Click and highlight the Legal Entity (if you have multiple)
3) In the Actions menu, click Set AMS Carrier Automation.

4) Check the box Allow Carrier Appt Automation and click save.

All warehouses linked to this same Legal Entity now has Scheduling Automation Activated.

5) Click on the Edit pencil icon for the Legal Entity
          a) Check box to allow carriers to edit appt.
          b) Check box to allow carriers to delete appt.
          c) Insert the number of minutes cut off window carrier has to make/edit appts. Outside of 24hrs                          in this example.
          d) Insert Default Last Load Time. Setting in time that system will allow last appts to be made.
