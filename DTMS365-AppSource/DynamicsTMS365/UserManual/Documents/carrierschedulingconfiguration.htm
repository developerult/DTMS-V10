Carrier Scheduling Configuration


Next to Contacts is a tab called Warehouse Carrier Settings. This tab contains configuration settings for carriers who are assigned to this warehouse and when the warehouse is not using the Automated Carrier Scheduling functionality at this warehouse. This feature will also override the Legal Entity default when automation is on and use carrier specific settings.

Note, when automated carrier scheduling is on at the Legal Entity, this means all locations and carriers are set to use automation. If you want to turn off automation for a warehouse, each carrier needs to be added here and turn off all features OR turn off the Validation on each dock door for all non-automated warehouses. Carriers will not be able to self-schedule appointments, but they will see the loads and can click a button and request an appointment. An email request will be sent to the scheduling administrator email indication carrier is requesting an appointment. As a reminder, this will need to be done for each non automated warehouse.

 

An example of what a carrier will see when trying to appoint and warehouse/carrier is not set for automation.

To assign carriers, click on ADD.

Select carrier and turn off all features as indicated in yellow highlight below. Click Save when done. This must be done for all carriers.

 

Additionally, when carrier automation is on and if you want to set carrier specific rules by location, users can apply carrier specific rules here.

Click on the pencil icon to edit carrier settings. When making changes to the carrier settings be sure to click save when done.

Note: If you have multiple locations using the scheduling automation system, all settings are by warehouse location and must be configured separately.

Carrier Appt = On/Off. When on, this carrier is allowed to self-schedule appointments in the scheduling system.

Allow Car Appt Edit = On/Off. When on, allows carrier to edit their appointments using the automated scheduling system. When off, it requires a request to be made via the carrier through the scheduling system.

Allow Car Appt Delete = On/Off. When on, allows the carrier to delete their appointment without any approval. When off, it requires a request to be made via the carrier through the scheduling system.

Appt Mod Cut Off Mins = The time in minutes setting that a carrier must appoint an order before a request needs to be made by carrier. For example, 1440 minutes means carriers must appoint orders before a 24-hr. window.

Default Last Load Time = The time designator for the last appointment available to be made. Meaning, the system will prevent a carrier using automation for scheduling appointments beyond this time. Carrier will need to submit a request for an appointment.
