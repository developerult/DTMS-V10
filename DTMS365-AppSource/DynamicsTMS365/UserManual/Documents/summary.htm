Summary

This is a list of the tasks to be completed when importing new or modified tariff rates.

1.  Create the Tariff Contract if it does not exist

2.  Create the Tariff Service/Equipment settings.  This will include the first row in the rate table.

3.  Edit the desired Break Points

4.  Open the TMS Desktop Application

5.  Go to System tab and open Tariff Import Tool

6.  Select your filters to load the data for the new or updated contract (created in steps 1 to 3 above)

7.  Ignore the Break points headers they are not displayed

8.  Export the template to the hard drive

9.  Open the template in excel

10. Steps 7, 8 and 9 are just to test that a file looks correct.  Once you have a valid template you can use this as a guide in the future for all similar tariffs (like LTL, Truck Load etc.).

11. It is recommended that you send the spreadsheet to the carrier and have them fill in the details.  This will minimize the amount of work needed in the next step.

12. Save the carrier’s response to the shared folder for tariffs. 

13. Check the file type and format.  If the file is a CSV file you will need to save it as an Excel file for example: Open the CSV file in Excel and save it as a Excel file with a name like:  CALODFLIN_CA ODFL IN-I_v-2.xlsx

14. Modify the columns in the Excel file provided by the carrier so it matches the template exactly.    See previously provided sample files as examples. 

15. Return to the Tariff Import Tool, in the TMS Desktop Application.

16. Check your filters and make sure you have entered the correct “Effective From” Date

17. Select “Load Rates From Excel”.

18. Wait while the rates load.  This can take several minutes depending on the number of rows and the speed of your computer.

19. Review the data on the screen.

20. Select “Import Rates to TMS Data”

21. This can take several minutes depending on the number of rows and the speed of your computer.  Not an hourglass or wait message may not be displayed.

22. Go back to the Tariff web Page and open the Rates.  You should see the new rates with a new revision number and the new Effective Date.
