Fuel

A carrier specific or shipper specific Fuel Matrix can be created for each contract. The system has a weekly update from the Department of Energy with the latest fuel pricing for that week. The system will calculate fuel based on load date and automatically update the fuel cost if a load was tendered say this week for next week. The fuel calculation at time of tender will show current DOE fuel calculation. TMS updates fuel costs from the DOE every Tuesday at 2am. API carriers do not have a fuel set up.

Changes to fuel surcharge can only be made if the ship/load date is in the past. Changes made before the load ships will be updated with the Carrier Fuel Settings.

NOTE:  The carrier fuel settings are pulled from the Fuel Addendum table and not necessarily from the Carrier Fees Table.

In the Action’s menu - Clicking the Fuel button brings up the Fuel Addendum form. This form is where information on fuel surcharge amounts are entered for this carrier contract.

 

To add a fuel matrix for a carrier, go to Contracts and highlight the carrier you want to add a fuel matrix for.

In the Actions menu, click on Fuel

The carriers Fuel Addendum page will open.

This section lists two possible selections for an average fuel cost source. They are:

•      Use National Average for Update = Turn ON

•      Use Carrier Default Fuel Price = Blank

 

 

 

 

 

Select an Exception Processing Method

 There are two options for handling possible exceptions:

•        Use National Zone Average for Each State = Turn ON (only if you intend to use zones like Midwest) otherwise turn off. You will need to configure your zones if turned ON.

•        Use Exception List for Specific States = Off (default)


Select A Revenue Calc Method

ON = Rate per mile
OFF = Percentage

 

Now you’re ready to import the fuel matrix from the Desktop Client.

Copy / Paste from spreadsheet

To copy from excel, the columns required are as follows:

Effective Date (date cells must be formatted as a date)
Low range (best to use two decimal places)
High range (best to use three decimal places to prevent overlapping)
Per mile cost
Percentage cost

Once you have your matrix in place, copy all the rows and columns.

Go to the Fuel Addendum page in the Desktop client for this carrier in Carrier Maintenance.

Place your cursor next to the PLUS icon (red below). To paste the information, use Control V.

 

Now go to the Contract page for this carrier, click on Fuel in the Actions Menu.

Users can copy an existing fuel matrix from a carrier in the desktop by going to the Actions menu and selecting Fiil From Template. This feature will copy the fuel matrix of the carrier you select and configured in the desktop client.

Click Fill From Template.

Use Specific Carrier for Template = Click ON.

Select the carrier from drop down list.

Click Save.

Click Refresh to see the data. Confirm your settings (per mile or percentage is correct). This copy feature is copying the fuel matrix from the desktop carrier maintenance fuel addendum screen.

To edit a fuel matrix, click the pencil icon to make your changes.

Anytime a change to the fuel matrix, users should run the Update All Active Loads feature.

The Fuel Addendum action menu:  Click the Update All Active Loads button to save the changes that you have made to the Fuel Addendum page. This is necessary to keep all freight costs updated. This is a long process and cannot be undone, so a warning box will appear. Click Yes to continue and the load information will be updated. NOTE: If load date and time is in the past, load will not be updated.

If you make immediate changes to a fuel matrix, this process will update all active loads.

To copy using the Global default carrier for fuel, leave the Use Specific Carrier for Template OFF.

Click Save.

Note: When costs are locked, the system will not update the fuel costs.

Note: Rate shopping only supports national fuel.
