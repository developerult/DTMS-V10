Configuration Parameters

DefaultCubeUOM (Default = CUI)
Default Unit of Measure for Cubes. Use CUI for cubic inches, CUFT for cubic feet, CCM for cubic centimeters, and CBM for cubic meters.

DefaultWeightUOM (Default = LBS)
Default Unit of Measure for Weight. Use LBS for Pounds and KG for Kilograms.

CarrierRequestAppointmentEmail (No Default value – must fill in for each Company that uses scheduler)
Email address to receive Carrier Appointment Requests. This is found in the parameters table under Company. Entering the email or group email address to manage carrier change requests.

If CarrierRequestAppointmentEmail is blank for a company this is the process that follows:

Check if this Company has any associated Company Contacts

If Contacts do exist, check for a contact with Notify checked and a populated email address

If no such contact exists, get the first contact with a populated email address

If still no email is found, check the Company Email address
