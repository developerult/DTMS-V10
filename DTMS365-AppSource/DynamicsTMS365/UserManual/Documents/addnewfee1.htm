Add New Fee

Go to the Accessorials Maintenance Page and click on Add.

Insert a fee name. Enter in EDI Code. The following variable will be used as defaults when configuring carrier specific settings for tariffs or for carrier automation.  Defaults may be replaced with a custom value for each tariff or lane fee as needed.

•      Minimum:  default value inserted into tariff or lane fees for the minimum charge. 

•      Variable:  This is the default cost variable used for the Variable Code calculation factor.  For example; when the Variable Code is set to weight the Variable value will be multiplied by the value in the weight for the order or load.    value will be used as the defaults when setting up or adding fees at the carrier level.

•      Auto Approve: select on or off to configure a default value for carrier freight bill processing automation in the Web Settlement page or via EDI 210 processing.  When off all fees entered by the carrier must be approved before the freight bill will pass audit, via the Fee Approval page.

The following settings are used globally by the system

•      Profile Specific – On, means this is a profile specific fee and can be assigned at the lane level or order level. Must add to each carrier fee set up in the tariff if you plan to use this fee as a profile fee.

•      Allocation Mode:   See Variable code section below for more details

o  None: not allocated, typically for order specific fees.

o  Load: allocate for all orders on the Shipment ID.

o  Destination: allocate for each matching delivery address (street address and zip code).

o  Origin: allocate for each matching pickup address (street address and zip code)

•      Allocation Formula: The factor, like weight, used to determine the percentage of allocation.

•      Calc Precedence:  See Variable code section below for more details

 

Accessorial fees are divided into Tariff, Lane, Profile, and Order specific fees. Tariff specific fees are associated with carriers. These fees are typically calculated in advance such as fuel or stops. Lane fees are location specific regardless of carrier. Profile Fees are fees that can be associated with a carrier’s tariff and assigned as part of the carrier selection process. Order specific fees are associated with an order and are typically fees determined upon delivery i.e. detention or unloading etc.  All accessorial fees may be applied to other orders on a shipment based on the allocation method of the fee.
