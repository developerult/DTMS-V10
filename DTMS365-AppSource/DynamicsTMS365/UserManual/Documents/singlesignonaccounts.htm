Single Sign On Accounts

This controls automated sign on connectivity to various API activity (such as p44 Account, NEXTStop or other API’s) by user.

To Rate shop using API’s you need to assign the p44 Account to each user. Like CSR’s and logistics users.

Find the user you just set up and click on the user, so it is dark blue.

Expand black arrow and click on ADD.

If a user is going to use API’s (for rating shopping/dispatching) and perform load postings, you will need to assign them to both p44 Account and NEXTStop Account. Assigning both to users are typically for logistics users only. This set up is a 2-part process. First, set up the user to use p44 Account, save information. Then go into the same user and assign the NEXTStop Account to the same user.

Account = Select either p44 or NEXTstop Account
(If selecting NEXTstop, just click save and you’re done. Other fields remain blank.)

Reference ID = This is the API group number (contact support if you do not know it)

SSOA Username = enter rramsey@nextgeneration.com

Password = NGL2016!

Expand the black arrow next to the user’s name to confirm to the Single Sign On Account set up.

Complete with having both SSOA’s for API and Load Posting.

The users are now set up and can use API rate shopping and Load Posting.
