Adding a Carrier User

 

In User Maintenance, adding carrier users will allow access to Web tenders, Load postings, Online scheduling, Freight bill entry. We have two types of carrier user set ups.

1) Carriers - This user profile is typically for a dispatcher (accepts loads, schedules appts, and provides load status.
2) Carrier Accountants – This user profile enters in freight bills into the website but also can be a dispatcher. This provide additional access to the Settlement/Settled screen for freight bill entry and review.

NOTE: If you have dispatchers who are responsible for entering in freight bill invoices, you can just set them up as a Carrier Accountants.

In the System page, click on User Maintenance.

In the Actions section, click on Add New Carrier User

 

Enter in carriers email address. This email address will be used in case carrier resets password.

Insert a username such as carrier users email address to keep it simple. johnsmith@carrier.com

You can pre-assigned passwords or have the system auto-generate one. It is recommended to manually create password for testing credential purposes. It is recommended that users test the carrier credentials prior to handing over username and password to carrier.

If Send Password to User is checked, an email will be sent to the carrier’s email address and include their password. Confirm receipt with carrier.

Check the box for Accounting ONLY if this user will be entering in freight invoices into the Web Portal. This will give the user access to the Settlement and Settled pages.

Select the carrier this user is associated with using the drop-down list.

Click Save in the upper right-hand corner.

All Carrier users will be assigned to the Legal Entity called NONE. This means the carrier will be able to see all loads if there are multiple legal entities.

Go to Legal Entity and select None. You will see a list of all carrier users.

Now, add this user as a contact in the Carrier Page. Check the default box if this contact is to receive all load tenders. It’s best to use the carrier group distribution email alias to communicate with multiple contacts.
