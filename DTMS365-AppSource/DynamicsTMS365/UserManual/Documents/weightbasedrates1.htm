Weight Based Rates

Can be configured as:

1) Country Code required
2) State rates
3) City rates (Country code and State/Prov required)
4) 3 digit or 5-digit (Start/Stop) postal code ranges (Country code required)
5) Lane specific-enter in the lane number from TMS Lane Maintenance for a rate to that destination.

LTL rates will require the use of weight brackets and carrier contracts can be configured separately according to the carrier’s specific weight breaks. In the example below, you have weight breaks starting at 1000, 2000, 5000, 10,000, 15,000, and 20,000. Anything 20,000 lbs. or more will use the 20,000 bracket.

All LTL rates will need to be converted to rate per pound rather than CWT (hundred weight).

After loading the carrier’s rates, be sure to confirm your weight brackets are set accordingly. The tariff system currently supports up to 10 bracket columns.

 

To edit the weight breaks, highlight a row in the rate page and in the action’s menu, click on Break Points.

Enter in the correct Break Points, click Save.

Click Rates, to review the break point settings are correct.

Class rates are required to have proper configuration and freight classes entered as part of the tariff. The system will only calculate the rate if the class is provided in the tariff and corresponding to the order.
