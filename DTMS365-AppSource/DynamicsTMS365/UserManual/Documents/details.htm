Once the contract is in place and the tariff type has been determine, TMS tariff template needs to be populated in preparation for import.

When importing rates, TMS requires the use of the standard TMS template provided as part of your implementation. This template is used for all rate type imports.

All rates need to have a country code assigned. If the country code does not match the country code found on an order, the system will not recognize that order and will display a message No Tariff Found when you attempt to rate and assign a carrier. Same if the spelling of a city does not match.

 

To import, go to the Systems Tab in the Desktop client and click on Tariff Import Tool found in the Systems tab on the desktop client.

The Tariff Import Management screen will pop open.

Begin by using the selection arrow to select the Carrier.

Select the warehouse location for the tariff assignment.

Select the most recent contract (revision number will display) you wish to update. (Note, the bottom one is the most recent and will display tariff name and Car Tar ID)

Select the Equipment associated with this contract.

Once you have selected the equipment, the system will auto-fill the tariff type.
- Rate Name
- Class Type
- Rate Type
- Bracket Type

Then enter in the effective date when you want this tariff to take effect.

NOTE: Once the new tariff is imported, the system will automatically place an Effective Date To on the tariff your replacing and insert an expiration date on it making it Inactive for historical purposes.

Next, click on #2 Load Rates from Excel. The file should be accessible in a shared folder or drive. After you click the button, the system will open a file folder menu for you to select the tariff file you want to import. Select the file and click Open.

The data will load in the tariff import screen for your review.

Then click on #3. Import Rates to TMS Data. Once it is complete, it will show a success message with the number of records imported.

The new tariff has now been loaded and users should review the contract/tariff.

NOTE-it is important to review the rates after import. You may have to update your weight break points or pallet breaks for any LTL tariff loaded.
