Carrier Fee Entry Setup


Once you have completed the carrier set up, you will now want to complete the carrier fees set up to enable carriers to enter their fee(s) and invoices depending on your TMS configuration.

In the Carrier Page click on the Actions Tab.

Click on Carrier Name so it turns blue.

Click Add Accessorials.

This is where you will set up all fees unique to this carrier and fees you expect them to enter into the system when entering invoices or for EDI transactions. Each configuration can contain a variety of fee(s). Most common will be fees such as:

          • Unloading    • Detention (Origin)  • Detention (Destination)

          • Tarp fee     • Re-delivery  • Stop charges                   • Fuel

• Misc (this is a for any fee that may not be configured but allows entry by carrier)

For each fee you enter, you must select the workflow configuration you desire by checking the boxes and click save when done.

Auto Approve = If fee is within tolerance, it will bypass manual/tms user approval if checked on. If not checked, users will have to approve this fee each time to pass audit.

Allow Carrier Updates = Allows carriers to enter this fee when entering an invoice if checked on. If not checked, carriers will not be able to enter fee.

Show Contracted Fees = Precalculated fees as part of the carrier’s expected cost and will display in the Carrier invoice entry screen when checked on. Recommended to leave checked. If not checked, carriers will not see fee and will not be able to change/update fee. Note, Carriers will not be able to edit fees.

Select your fee from the drop-down list. These fees are assigned in the Carrier page for each carrier. It is important to configure and make available all fees to carriers you expect them to enter. If a fee does not appear here, the carrier will not see the fee to apply for their invoice. Web Settlement Carriers should contact Logistics to set up fee if a fee is missing prior to entering invoices.

Once your fee is selected from the drop-down list, the Caption field and EDI Code (if set up) will auto populate. Users can insert caption text description if missing for the accessorial your configuring. Users should not modify EDI Code field unless they feel and have confirmed it is incorrect per carrier EDI Set up.
