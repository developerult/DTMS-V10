Add New Contract

When creating a new contract, the following items are required to complete a carrier contract/tariff set up and be ready to rate out shipments.

1) Create contract.
2) Need to add the Service/Equipment (LTL, FTL, Reefer, Van etc. most come preconfigured)
3) Need to apply any Exceptions (Discounts, Minimums, etc.) applies to LTL.
4) Need to import carrier Rates according to tariff.
5) Need to configure your break points (weight breaks or pallet breaks for LTL).
6) Need to configure the Fuel Matrix.
7) Need to apply any carrier Fees (stop charges, fuel, taxes etc.)
8) If carriers are to be entering in freight invoices in the portal or if using EDI, see TMS CARRIER document for additional freight audit tolerance set up.

 

Go to Contracts create a new Carrier Contract. The system will take you to the carrier contracts page.

Click on ADD

Required fields are as follows:

1.       Tariff Name = Name (Examples: RidgeWay - TL, Swift – Intermodal, FedEX – LTL Economy

2.       Carrier Number = Select carrier for the new contract

3.       Select the company/warehouse location for this contract

4.       Car Tar ID (Example:  Contract Number or another unique format)

3.       Effective Date when the contract starts (Required)

4        Effective Date when the contract ends (not required)

5.       Def Weight - Check on if rating requires deficit weight calculations for LTL.

6.       If Outbound leave ON. If Inbound turn OFF.

7.       Drive Saturday = On/Off     (On for TL Off for LTL)        Drive Sunday = On/Off (On for TL Off for LTL)

8.       Mode Type = Select mode (typically road). Use Rail for Intermodal.

9.       Tariff Type (Private = negotiated contracts, Public = public contracts)

10.     Temp = Dry, Frozen, Cooler, Any (must match service/equipment being used for this contract)

11.     Auto Assign Pro = Turn off if not configuring to auto generate carrier Pro Number Logic.

 

Once you have created a contract, you now need to assign the Service to the contract.

Find the contract you just created and click on the row, so it is highlighted.

In the Actions menu select Service. Note, Service is synonymous with Equipment.

Examples are:

LTL
Truckload
Reefer
Dry Van
BTrain

 

 

 

The Tariff Service Data screen will open.

The highlighted fields are required.

Equipment = Select the appropriate service/equipment as part of the system configuration. For
                        example, Van or Reefer or LTL or FTL. Equipment defaults should already be configured
                        based on your implementation.

Temp = Temp must match equipment added as part of system configuration. This should default to the
               correct temperature.

Bracket Type = The bracket calculation method for this tariff.
                             LTL = Select the bracket type that matches your tariff. Such as Lbs. or Plts (pallets). CWT
                             is not supported and should be converted to Rate per Pound when importing rates.
                             Truckload = If creating a truckload tariff, skip this setting.

Class Type = Freight class for this tariff.
LTL = Most configurations will be FAK
Truckload = Skip this step

 

Rate Type = The rate type for this tariff.
Class = If this is an LTL tariff.
Distance = If this is a Rate per Distance/Mile.
Flat = If this is a Flat rate.
Unit of Measure = If this is a per pallet rate etc.

 

Rate Name = Examples can be LTL FAK 70 or Distance (for rate per mile tariff) or Flat (for flat rate).

Rate Country = Entering the country code will seed the first row for entering in a rate to generate a
                             tariff. You do not have to fill in the State/Prov, City, or postal code ranges at this time.

 

Example 1 = Truckload Rate Per Mile Tariff (Note: Flat rate would be the same, but Rate Type should read Flat). The system will use the default capacity settings for each equipment.

Users can modify the capacities associated with the default Service settings. For example, the default Wt. Max is 45,000 lbs. If you want to change this to 20,000 lbs. you can do so. This means that this tariff will be made available for any order/load under 20,000 lbs. 

 

Example 2 = Pallet Based Tariff

Users can modify the capacities associated with the default Service settings. For example, the default Wt. Max is 45,000 lbs. If you want to change this to 20,000 lbs. you can do so. This means that this tariff will be made available for any order/load under 20,000 lbs. 

Example 3 = LTL Class/Weight Based Rates

 

Users can modify the capacities associated with the default Service settings. For example, the default Wt.. Max is 45,000 lbs. If you want to change this to 20,000 lbs. you can do so. This means that this tariff will be made available for any order/load under 20,000 lbs. 

Once you have completed the Contract and assigned the Service/Equipment set up, the system automatically creates the first version of a tariff matrix for you. If using Break Points for LTL, set up your weight breaks by highlighting the first row of your rates and click on Break Points to insert the brackets. Now you can import your rates. Please review your break points following import.

Note: User must perform additional set up for LTL Class Rates following rate import. See Class Exceptions, Discounts, Minimum Charge configurations for LTL. If applicable.
