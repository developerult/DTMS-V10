Carriers Accessorial Management

 

The system comes with a set of pre-configured accessorial fees. Users should first review the accessorial fees to determine if a particular fee can be used rather than adding a new one.

For example, Unloading Fee is the same as Lumper Fee. The Accessorial page is where users go to add new accessorial set up if one does not exist or can’t use a pre-configured accessorial fee.
