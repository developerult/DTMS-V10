Class Exceptions

Click on the contract you want to configure Class Exceptions, so it is highlighted.

In the Actions menu, click on Exceptions

The Class Exceptions screen will open.

Click on Add

Class exceptions can be added and managed within the TMS system. Enter in the following information:
Name: FAK70 (General name that defines this exception)
Actual From: Class 50
Actual To: Class 100
Rating As: Class 70
Point Types: Direct, Interline, or Any
Class Type: FAK

This above example means that the system will use class 70 for any rate shop/order with a class between Class 50 – Class 100.
