Appointment Status Color

Appointment Status Color section allows the user to add, edit and delete appointment status color for the selected warehouse.

Add Appointment Status Color

1.  Admins can add a new appointment status color for the warehouse by clicking on “+Add Appt Status Color” available in the top toolbar of appointment status color grid.

2.  When the button is clicked the grid displays the new row with input fields as seen in the screenshot below.

3.  Once the details are entered, admin should click save  button to add the new appointment status color or they can click clear  button to cancel adding the new appointment status color.

Edit Appointment Status Color

1.  The admin can edit details of an existing appointment status color by clicking on edit  button available in each row of appointment status color list.

2.  They can click on edit button that needs to be edited in left pane the columns will be displayed in edit mode to modify details.

3.  Once the details are modified, the user should click on save   button to save the changes or click cancel   button to cancel the changes.

Delete Appointment Status Color

1.  The user can delete an appointment status color by clicking on Delete   button on the particular row that needs to be deleted in the list.

2.  A confirmation dialog box will be displayed asking the user to confirm deletion of the record.

3.  Click “OK” to delete or “Cancel” to cancel deleting the row.
