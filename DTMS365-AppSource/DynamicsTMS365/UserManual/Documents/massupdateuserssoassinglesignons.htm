Mass Update User SSOA’s (Single Sign On’s)

This feature allows Admins to assign SSOA’s to multiple users in a single instance.

NEXTStop-Mass Update

Click on the Action Menu button “Mass Update User SSOAs”


Check the “NEXTStop” checkbox to create a NEXTStop Account
Select all the users you wish to add NEXTStop accounts for by checking the boxes in the grid



Click the “save” button in the window tool bar

Note: This function only adds the NEXTStop Account to the selected users if they do not already have one. It will not create duplicates if a user already has a NEXTStop SSOA. It will not delete the NEXTStop Account for the users who are not selected in the grid.

NGLAPI -Mass Update

Click on the Action Menu button “Mass Update User SSOAs”



Check the “NGL API” checkbox to create a P44 Account
Select all the users you wish to add P44 accounts for by checking the boxes in the grid



Click the “save” button in the window tool bar

Note: If a user already has a NGLAPI Account, it will overwrite/update the SSOA User Name, Account Group (Reference ID), and Password. It will not delete the NGLAPI Account for the users who are not selected in the grid.
