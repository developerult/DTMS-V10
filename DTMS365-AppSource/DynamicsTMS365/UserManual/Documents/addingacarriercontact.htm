Adding a Carrier Contact


Go to the Carrier Page and find the carrier you want to add a contact to.

Expand the black arrow and click on Contact.

Click on the Add button to insert the new carrier user contact information. The email address when populated will be the email address used to tender loads to. If blank, the system will not send an email tender to the carrier.  If default is checked, this contact email address will be the dispatch/tender default used to send tenders for this carrier. It is recommended to use a carrier’s distribution group email address for tenders rather than individuals’ emails addresses. If needed, you can insert multiple email addresses followed by a comma in the same default email address field. When adding a contact here, it also updates the desktop client.

Dispatch Default = When ON, the system will use this email address when sending email dispatch tenders to carriers. If no email is configured, the carrier will not receive and email notification.

When OFF, users will need to assign the correct dispatch contact upon dispatching the load to carrier. Note, API carriers will not receive an email for tenders by default.
