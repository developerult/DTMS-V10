1) Quick Edit-

Where Carriers or TMS Users only want to enter the total invoice amount.

Quick Edit

Click on the Quick Edit button in the Actions menu. A dialog box will open.

Insert the Invoice Number

Insert the Invoice Amount – This should be the total amount of the invoice including all fees.

Expected Amount- This is what was estimated at booking.

Users can add any number of invoices at one time and when done. Click SAVE.

The system will display a success message for all saved records. Any records not invoiced will remain open and a warning message appear indicating the remaining loads. Click on warning message and it will disappear.

The Status column will indicate the status of each invoice.

N Status = Are new records waiting for freight bill entry.

PENDING Status = waiting on TMS user to approve entries.

FAILED Status = Are freight bills that have been entered but did not pass the company level tolerance audit function. This requires a TMS user to review this exception. Often times it could be a fee has not been inserted on the order. Users should run the Audit report to see the difference and or review the load and perhaps update the load with any missing fees. Then re-run the audit.

 

Click on the Information icon for to review the pending/failed freight bill entry. Note you will not be able to edit the freight bill after it has been entered and saved.
